Fireque Cli
=======
Fireque 提供cli的操作界面


## Option
`-h -host`
`-p -port`



## Keeper
```
fireque keeper [option]
```
> 顯示 HIGH, MED, LOW 各幾筆, 送至Queue


## Monitor
```
fireque monitor [option]
```
> 以protocol做為group

> 顯示 queue, processing, completed, failed, timeout 數量

> 顯示buffer, schedule數量



## Client
```
fireque client [option]
```
> ls

> new

> get
